## Val/test post-processing asymmetry (new finding)

This is the main M2.4 "Layer 3" finding — not in paper:

| | Val (downsampled) | Test (natural) |
|--|-------------------|----------------|
| dayofyear==1 rows | **0** (downsampling excluded them) | 192 rows |
| dayofyear>366.9583 | 2 | 206 rows |
| meter_reading==1.0 | 6,528 | 17,660 |

Rule 2a/2b's effect is **only visible on test set**. Val shows nearly zero ΔAUC from post-processing (ΔAUC = +0.0004, within noise floor). Paper §2.4 describes post-processing as if it's generally useful, but the val-side evidence doesn't support this — it's test-set targeted.
