# Handoff: M4.1 Importable Pipeline Foundation Complete

**Date**: 2026-06-24
**Status**: Complete - behavior-preserving refactor passed golden gates
**Scope**: M4 planning, M4.0 baseline lock, M4.1 `src/lead` extraction

## Summary

M4 planning and the first two implementation slices are complete. The repo now
has an importable `src/lead` package for the M3 foundation helpers, and the
tracked M3 experiment scripts import that package instead of carrying local
copies of the core helper functions.

The refactor intentionally preserves current M3 semantics:

+ M3 label assignment is still positional; M4.2 owns key-aligned join.
+ Value-change features still use row-offset `groupby().shift()`; M4.3 owns
  timestamp-merge semantics.
+ `StandardScaler` and the `[negs1, pos, negs2, pos]` downsample pattern are
  preserved; M4.4 owns any cleanup.
